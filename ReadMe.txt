# EcoCryptoGuide Chatbot

A simple command-line chatbot that provides eco-friendly cryptocurrency advice and information.

## Description

EcoCryptoGuide is an interactive chatbot that helps users learn about cryptocurrencies with a focus on sustainability. The bot maintains a database of popular cryptocurrencies and can provide information about:

- Sustainable cryptocurrency options
- Trending cryptocurrencies
- General crypto advice

## Usage

1. Run the bot using Python:
```bash
python bot.py
```

2. Interact with the bot using commands like:
- "What's the most sustainable crypto?"
- "Show me trending coins"
- "Help"
- Type "quit", "exit", or "bye" to end the conversation

## Requirements

- Python 3.x
- No additional packages required

## Data Structure

The bot includes information about three cryptocurrencies:
- Bitcoin
- Ethereum
- Cardano

Each cryptocurrency entry contains:
- Price trend
- Market cap
- Energy usage
- Sustainability score
